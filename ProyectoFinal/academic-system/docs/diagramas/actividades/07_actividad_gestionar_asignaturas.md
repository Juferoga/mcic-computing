## 3. Reglas de Negocio Aplicadas

1. **CГіdigo Гәnico**: No pueden existir dos asignaturas con el mismo cГіdigo
2. **Nombre obligatorio**: El campo nombre no puede estar vacГӯo
3. **Intensidad horaria**: Debe ser un nГәmero entero positivo (0-20)
4. **CrГ©ditos**: Debe ser un nГәmero entero positivo (1-10)
